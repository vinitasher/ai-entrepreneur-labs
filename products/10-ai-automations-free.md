## Why Automate?

Every hour you spend on repetitive tasks is an hour you're not spending on what actually matters. These 10 automations are real, tested, and cost little to nothing to set up. Most take under 30 minutes.

## 1. Auto-Reply to Common Customer Emails

**Tool:** Gmail + Make.com (free tier)
**Time to set up:** 15 minutes
**Saves:** 3-5 hours/week

Set up a filter that detects common questions (pricing, shipping, returns) and drafts a personalized reply using AI. You review and send — no more typing the same email 20 times.

**How:**
1. Go to make.com → Create scenario
2. Connect Gmail → set trigger "New email matching search"
3. Add AI module → feed it your FAQ document
4. Output: Draft reply in Gmail, ready to review

## 2. Social Media Content Repurposing

**Tool:** ChatGPT + Buffer (free tier)
**Time to set up:** 20 minutes
**Saves:** 5-8 hours/week

Write one piece of content. AI rewrites it for Twitter/X, LinkedIn, Instagram captions, and email newsletter. One effort, four platforms.

**How:**
1. Write your core post/article
2. Prompt: "Rewrite this for [platform] in [tone], under [character limit]"
3. Schedule all versions in Buffer
4. Post consistently without the grind

## 3. Meeting Notes → Action Items

**Tool:** Otter.ai + ChatGPT
**Time to set up:** 10 minutes
**Saves:** 2-3 hours/week

Otter transcribes your meetings. Feed the transcript to ChatGPT with: "Extract action items, owners, and deadlines from this meeting transcript. Format as a checklist."

**How:**
1. Enable Otter for your Zoom/Google Meet calls
2. After meeting, copy transcript
3. Use this prompt template:
```
Extract from this meeting transcript:
- Action items (what needs to be done)
- Owner (who's responsible)
- Deadline (if mentioned)
Format as a markdown checklist.
```

## 8. Customer Feedback Analysis

**Tool:** ChatGPT + CSV upload
**Time to set up:** 10 minutes
**Saves:** Hours of manual categorization

Export reviews/feedback → AI categorizes sentiment, extracts themes, and ranks issues by frequency. Know what to fix first.

**Prompt template:**
```
Analyze this customer feedback data. For each entry:
1. Sentiment (positive/neutral/negative)
2. Category (product, support, pricing, UX, other)
3. Key issue or praise

Then provide:
- Top 5 issues ranked by frequency
- Overall sentiment score
- 3 recommended actions
```
